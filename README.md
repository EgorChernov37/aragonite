## How to use?
### UNIX
- Open terminal using <kbd>Ctrl</kbd> <kbd>Alt</kbd> <kbd>T</kbd> and type in prompt
    ```sh
    python3 shell.py
    ```
### Windows

- Install Python with that tutorial

- Press <kbd>Win</kbd> <kbd>R</kbd>
- Type "cmd"
